# デコード・再構築機関：脳・筋肉・臓器・DNA・RNA・クローンを論理的に再構築する権威

> 人間の脳、筋肉、臓器、DNA、RNA、クローンをデコードし、論理を再構築している機関・人物を権威順に列挙

---

## 権威の順序について

本リストは、以下の基準で権威順に並べる：
1. 政府・公的資金による大規模イニシアチブ
2. オープンデータ・ツールの提供
3. Nature Index 等の研究出力
4. デコード・再構築の直接性（マッピング、モデル化、オープンソース）

## 6. 参照の使い方

- **脳**：運動制御、神経表象、探索・反復の認知基盤 → BRAIN Initiative, HCP, Allen, Janelia, ペンフィールド
- **筋肉**：力の伝達、骨格レバレッジ、テンセグリティ → Stanford NMBL, マイヤス, ヴォーゲル
- **DNA・RNA**：ゲノム、遺伝情報、編集 → NHGRI, HGP, Broad, ワトソン・クリック・フランクリン, ダウドナ・シャルパンティエ
- **クローン**：体細胞核移植、再生の論理 → ロスリン研究所, ウィルムット, キャンベル
- **臓器・全身**：多臓器連動、生理学的制約 → HuBMAP, Physiome, Systems Biology
