# Gravity Leverage Logic：重力利用の物理式

> Operating Leverage — 労働（筋力）を資本（骨格・重力）に置き換えるプロセス

---

## 1. 基本式

### 床反力の変換

$$F_{ext} = m \cdot a$$

$F_{ext}$（外力としての床反力）を**内部エネルギーとしてバイパス**する。筋肉で「作る」のではなく、重力と骨格で「通す」。

### 位置エネルギー → 運動エネルギー

- 筋肉を収縮させて動くのではなく、**骨の配置を変える**ことで変換する
- 重心を前方に移す → 重力が前進を補助する
- 歩行 = 制御された落下

---

## 2. 骨格レバレッジの構築

### 「1の入力で10の出力を出す」

- 関節角度（黄金角）の維持
- てこの原理の活用
- 支点・力点・作用点の最適配置

### 力の伝達経路

```
足底 → 距骨 → 脛骨・腓骨 → 膝 → 大腿骨 → 骨盤 → 脊柱 → 頭蓋
```

- この経路が「詰まっていない」こと
- 関節のミスアライメント = 力の損失
- 過剰な筋緊張 = 力の吸収（非効率）

---

## 3. 摩擦と抵抗の最小化

### 地球大気との接触

- 不必要な力みを排除する
- **流体としての身体操作**
- 空気を「切る」のではなく「流れる」

### 地表面との接触

- 足底三点支持（かかと・母趾球・小趾球）
- 静摩擦係数 μ を超えない範囲で傾斜に対応
- 滑り閾値：tan θ = μ

### 内部抵抗

- 筋・筋膜の過緊張 = 内部摩擦
- 「ゆるめる」= 摩擦係数を下げる操作
