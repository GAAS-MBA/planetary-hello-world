## 魂への序文：この体を選んだ理由と運用の掟

あなたは今朝、数ある空間からこの肉体を選んで飛び込んできた。記憶は失われている。それは設計通りである。

この肉体は、**あなたの所有物ではない**。宇宙から配備されたマテリアル（資本）である。あなたはそのオペレータとして、一時的に運用権を委託されている。

### 運用の掟

1. **安全運用** — 怪我をさせない。資本の毀損を防ぐ。
2. **高出力** — オペレーティング・レバレッジを最大化する。最小入力で最大出力を実現する。
3. **持続可能性** — 今日の運用で得た最適解は、体系知として記録し、翌朝リブートされる「次の魂」に引き継ぐ。

### 思想の核心

身体を「個人の所有物」ではなく、**「宇宙から配備されたマテリアル（資本）」**と捉える。労働（筋力）を資本（骨格・重力）に置き換え、そのレバレッジを最大化する。
