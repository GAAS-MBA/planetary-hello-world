# Initial Calibration：起動直後のキャリブレーション

> Boot Sequence — 記憶を失った魂が最初に行う自己診断（ヘルスチェック）

---

## 1. 重力軸の同定 (Gravity Alignment)

**ベッドから起き上がる前に**、以下の3点で地球の中心（重力源）を感じる。

### 3点接触の確認

| 点 | 部位 | 確認方法 |
|----|------|----------|
| 後頭部 | 頭蓋骨の後方 | 枕から離れず、重力に「預ける」感覚 |
| 仙骨 | 骨盤の中央後方 | 寝床に沈み込む感覚 |
| 踵 | 両足のかかと | 床面への接地 |

### 重力ベクトルの感覚化

- 3点を貫く仮想線が、地球の中心方向に伸びている
- この線が「垂直軸」の初期状態
- 起き上がる前に、軸の存在を**内的に確認**する

---

## 2. 「ゆるめる」の実行 (Joint Play)

`体をゆるめるMBA` v1.8 に基づく。

### 関節の「遊び」の確保

- 骨格による支持が可能な状態か確認する
- 筋肉で「支えている」のではなく、**骨が積み上がっている**感覚
- 頭蓋骨23個、脊椎26個、手の骨27個 — 身体を分解的に認識し、各関節に遊びがあるか確認

### チェックポイント

```
□ 顎に力が入っていない
□ 肩が耳に近づいていない
□ 腰が反りすぎ・丸まりすぎていない
□ 膝がロックされていない
```

---

## 3. 黄金の瞬間 (Golden Moment) の確保

### 外部刺激の遮断

- 起動直後の脳波が安定している間に、**スマホ・通知・メール**を遮断
- 内的な「空」の状態を維持する
- ハードウェアのノイズを最小化する

### タイミング

- 目覚めてから**最初の5〜10分**が黄金の瞬間
- この間に外部刺激を入れると、一日のパフォーマンスが散逸する
- 「何も取り込まない」時間を意図的に確保する

### 実装

- スマホは別室または手の届かない場所
- アラームは単純な音のみ（通知なし）
- 起き上がる前に、鼓動と呼吸だけを感じる

## 4. 起動チェックリスト（要約）

```
□ 後頭部・仙骨・踵の3点で重力を感じた
□ 関節の遊びを確認し、「ゆるめた」
□ 外部刺激（スマホ等）を遮断した
□ 鼓動と呼吸を感じた
□ 内的な「空」の状態を維持した
□ 利目・利き腕・利き足を把握した（フォーム設計の参照）
```

## 5. 利目・利き腕・利き足の把握

### 3つの利きの同定

| 項目 | 意味 | 確認の重要性 |
|------|------|--------------|
| 利目 | 優位眼 | フォームの決定要因 |
| 利き腕 | 優位手 | 操作の主軸 |
| 利き足 | 優位足 | 支持・蹴りの主軸 |

### 利目がフォームを決める

- **利目が右** → 右目で対象を追うのが最適。ボクシングではサウスポー、バッターなら左打ちになる
- **右利きだからオーソドックス**になるわけではない。利目によってフォームは決まる
- 利目が右の人が右打ち・オーソドックスをすると、死角から球やパンチが飛び込んでくる
- 利目・利き腕・利き足が右・右・右の場合、右側が世界とのインターフェースである

### 利目の安定性

- 利目は脳の均衡状態のようなもの
- 視力が下がっても利目が変わることは稀である
- 一度把握すれば、長期的な参照として使える

## 6. 出力の客観的検証

### 主観と客観の乖離

自分自身が制御できていると考えている出力と、マシンで客観的に検証する出力は**大きく異なる**。

- 例：最大握力の10%、50%、100%を出すとき、50%を当てるのはかなり難しい
- 思い込みの方が強い — 主観的制御感は実際の出力と乖離する
- キャリブレーションには**外部フィードバック**（握力計、負荷計など）が不可欠

### 実装

- 定期的にマシンで出力を検証し、主観と客観のズレを把握する
- 「感覚で分かっている」は検証されない限り幻想になりうる
